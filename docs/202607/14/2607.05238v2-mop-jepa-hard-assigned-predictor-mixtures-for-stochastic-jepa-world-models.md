## 摘要
JEPA世界模型通常使用单个回归器预测下一个潜在状态。在随机转换下，平方回归和余弦回归分别返回条件均值及其归一化方向：一个可能无法匹配任何有效后继状态的折中方案。MoP-JEPA 则使用 K 个硬分配的头和一个仅上下文的路由器，在单次前向传播中生成一个有限的候选集。在保留的 OGBench 转换上，使用单输出预测器的图搜索在 0.02–0.09 的查询上成功，而 MoP-JEPA 达到了 0.85。为了区分有用的后继状态与无差别的覆盖，我们还测量了验证路径成功率（realroute），该指标在图构建后检查提案是否包含一条真实转换的路径。MoP-JEPA 在所有三个迷宫的相同协议指标上领先；MDN 获得了较高的原始覆盖率，但预测了许多不存在的边。

## Abstract
JEPA world models commonly predict the next latent state with one regressor. Under stochastic transitions, squared and cosine regression return the conditional mean and its normalized direction, respectively: a single compromise that may match no valid successor. MoP-JEPA instead uses $K$ hard-assigned heads and a context-only router to produce a finite candidate set in one pass. On held-out OGBench transitions, graph search with single-output predictors succeeds on $0.02$--$0.09$ of queries, whereas MoP-JEPA reaches $0.85$. To distinguish useful successors from indiscriminate coverage, we also measure verified-route success (\emph{realroute}), which checks after graph construction whether the proposal contains a path of real transitions. MoP-JEPA leads this same-protocol metric on all three mazes; an MDN attains high raw coverage but predicts many nonexistent edges.

### 论文核心问题与整体含义（研究动机和背景）

传统的JEPA世界模型在预测下一个潜在状态时使用单个回归器（如平方损失或余弦损失）。在随机转换（stochastic transitions）下，一个上下文可能对应多个有效的后继状态，而单个回归器只能输出条件均值或其归一化方向——这个折中值可能无法匹配任何真实的后继，导致下游规划（如图搜索）失败。现有方法如加权混合专家（MoE）依然输出单一向量，不能解决多模态问题。MoP-JEPA旨在通过硬分配预测器混合（hard-assigned predictor mixtures），在单次前向传播中生成有限候选集，从而保留多模态信息，提升随机场景下的规划可行性。

### 论文提出的方法论

**核心思想**：用K个独立的预测头（predictor heads）替代单个回归器，每个头学习一个候选后继；同时引入一个上下文路由器（context-only router）预测每个头的激活概率。训练时，每个目标样本只更新距离最近的头（硬分配），路由器学习从上下文预测该索引。推理时，模型输出K个候选潜状态及其路由器权重，供图搜索或MPC使用。

**关键技术细节**：
- 编码器`fθ`和EMA目标编码器`fξ`不变（保持JEPA主干）。
- 输入上下文`c = [z, a]`，其中`z`是归一化的当前状态潜码，`a`是动作（动作自由场景中`c=z`）。
- K个预测头`gϕk`输出`uik`，路由器`rψ`输出概率`πik`。
- 训练损失包含三项：硬分配余弦距离损失（仅更新获胜头）、路由器交叉熵损失（监督信号为获胜头索引）、负载平衡KL散度（惩罚头使用不均匀）。
- 推理时，使用所有K个头或通过路由器阈值选择子集作为候选集。

**公式/算法流程**（文字说明）：
1. 编码当前状态`s_i`，得到归一化潜码`z_i`，组合上下文`c_i`。
2. 将`c_i`分别送入K个预测头，得到候选`u_ik`；送入路由器得到概率`π_ik`。
3. 对目标`z_i^+`，计算与每个候选的余弦距离，选择最近的作为获胜头`k*`。
4. 更新损失：获胜头距离（只反向传播到该头）+ 路由器交叉熵（标签为`k*`）+ 负载平衡项。
5. 推理时输出集合`{ (u_k, π_k) }`。

### 论文的主要结论与发现

1. **单输出限制**：密集/门控MoE预测器输出条件均值，在随机分支处位于无效中点，导致图搜索成功率极低（0.02–0.09）。
2. **MoP-JEPA有效性**：在保留的OGBench转换上，planAll达到0.85（分别对比密集0.08、M3-JEPA 0.075、Var-JEPA 0.085、MDN 0.566）。realroute在三个迷宫中均领先MDN（如medium-stitch: 0.247 vs 0.134）。
3. **模式恢复**：MoP-JEPA能恢复82–88%的三/四路分支后继，而密集/门控模型恢复率为0。
4. **转移精度**：MDN高覆盖率伴随低精度（0.14–0.21），而MoP-JEPA精度为0.42–0.56，表明其预测的真实边更有用。
5. **长程规划**：在距离≥6的远程目标上，MoP-JEPA保持非零realroute，而密集和MDN迅速降为零。
6. **跨域泛化**：在ETH/UCY、SVHN、DINO-WM等任务中，MoP-JEPA表现优于密集，但在部分高维场景增益缩小。

### 优点

1. **方法新颖简洁**：将硬分配多候选学习引入JEPA世界模型，仅修改预测模块，保持编码器不变，易于集成。
2. **验证协议严谨**：引入realroute和转移精度等控制指标，区分了“覆盖噪声”与“有用后继”，避免了对高覆盖率指标的过度信任。
3. **消融与控制实验丰富**：包括码本、打乱上下文、路由器门控、K扫描、头分离损失等，全面验证了设计必要性。
4. **跨域验证**：不仅在低维迷宫，还在行人预测、图像修补、像素观测等场景进行了测试，增强了结论的普适性。
5. **公平对比**：所有方法使用相同架构、数据、规划器，避免了不公平的优势。

### 不足与局限

1. **种子的双峰性**：在硬迷宫（large-stitch）上，MoP-JEPA的WTA训练出现seed bimodality，部分种子性能差，需通过router熵选择好的初始化，但规则仅在该任务族验证。
2. **稀有模式丢失**：在四路分支中，概率低于10%的后继可能未激活，仅37%的此类单元完全恢复四个模式。退火或头排斥未能解决。
3. **高维场景增益有限**：在蚂蚁迷宫（29维）和像素观测中，MoP-JEPA的raw覆盖率提升，但门控覆盖率和realroute增益弱于低维迷宫；在DINO-WM端到端MPC中，统计上未显著改善闭环执行。
4. **未用于大规模场景**：视频级或机器人级世界模型（如V-JEPA 2）的验证未进行，论文仅指出确定性预测器可能在随机分支处产生折中，但未在真实机器人任务中测试。
5. **资源消耗未报告**：未给出训练MoP-JEPA的计算成本（额外K个头和路由器），可能比单头预测器开销更大。
6. **规划器依赖**：realroute评估依赖图搜索，而闭环执行可以修复部分低精度预测，导致执行成功率对模型保真度不敏感（MDN执行成功率高但realroute低），说明需要更细粒度的评估协议。

### 总结与展望

MoP-JEPA通过硬分配混合专家预测器，为JEPA世界模型赋予了生成多模态候选后继的能力，有效解决了传统单回归器在随机分支处输出无效中点的问题。实验表明，该方法在低维离散控制任务（如点迷宫）中显著提升了图搜索成功率和路径保真度，且通过路由器实现了可解释的、可操作的候选排序。然而，其在更高维或像素级观测任务上的增益有限，且存在种子双峰性、稀有模式丢失、计算开销未量化等不足。未来工作可探索：① 引入自适应头数（如动态K的剪枝或生长策略）；② 结合软硬退火渐进训练以稳定稀有模式恢复；③ 将MoP集成到端到端MPC中（如DINO-WM），并评估其在真实机器人环境中的闭环性能；④ 分析计算-精度权衡，降低额外开销以支持大规模视频世界模型。总体而言，MoP-JEPA为学习随机环境下的世界模型提供了简洁而有效的多候选范式，其设计思路对后续不确定性感知的潜空间预测研究具有启发意义。
