## 摘要
我们提出了FabriVLA，一种用于精确多任务操纵的轻量级视觉-语言-动作模型。FabriVLA结合了InternVL3.5视觉-语言骨干网络与一个流匹配动作头，该动作头具有跨动作令牌的门控自注意力机制以及浅层VLM层融合以丰富空间上下文。该模型通过从预训练VLM和随机初始化的动作头进行单阶段联合优化来训练。在涵盖50个不同操纵任务的Meta-World MT50基准上，FabriVLA达到了90.0%的分层平均成功率，表明基于1B规模VLM构建的紧凑型VLA无需依赖数十亿参数的VLA骨干网络即可实现强劲性能。

## Abstract
We present FabriVLA, a lightweight Vision-Language-Action model for Precise Multi-Task Manipulation. FabriVLA combines an InternVL3.5 vision-language backbone with a flow-matching action head featuring gated self-attention across action tokens and shallow VLM layer fusion for enriched spatial context. The model is trained via single stage joint optimization from a pretrained VLM and randomly initialized action head. On the Meta-World MT50 benchmark spanning 50 diverse manipulation tasks, FabriVLA achieves a tier-average success rate of 90.0%, demonstrating that a compact VLA built on a 1B scale VLM can achieve strong performance without relying on multi billion parameter VLA backbones.

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：构建一个轻量级的VLA模型，保留一个1B规模的预训练VLM骨干（InternVL3.5-1B），并设计一个基于流匹配（flow matching）的动作头，通过**门控自注意力**（gated self-attention）跨动作令牌建模时间依赖，以及**浅层VLM层融合**（shallow VLM layer fusion）来注入更丰富的空间细节，从而在不增加大量参数的前提下提升操纵精度。

- **关键技术细节**：
    1. **整体架构三组件**：
        - InternVL3.5-1B视觉-语言骨干（保留14个transformer层），将448×448的图像和任务指令编码为上下文令牌。
        - 状态/动作编码器：机器人状态（24维）通过MLP编码为状态令牌，动作（50步×24维）通过共享MLP和位置编码编码为动作令牌序列。
        - 流匹配动作头（flow-matching action head）：由8个堆叠的Transformer块构成，每个块包含门控自注意力、交叉注意力（到VLM上下文）、带时间条件的FFN。
    2. **门控自注意力**：
        - 在动作头每个块的开始处，通过可学习门控参数g（初始化为0）控制自注意力：A' = A + g·SelfAttn(LayerNorm(A))。初始化时门控关闭，行为如同仅交叉注意力；训练时门控逐渐打开，自动学习动作令牌间的时序依赖。这提供了平滑的优化路径。
    3. **浅层VLM层融合**：
        - 同时提取VLM深层（第14层）和浅层（第6层）的特征，通过可学习线性投影（2048→1024，初始化为[I|0]）将两者拼接融合，输入动作头。浅层特征保留了更丰富的空间细节（如物体边界、相对位置），对精确定位和接触式操纵至关重要。融合后仅增加约210万参数。
    4. **流匹配训练**：
        - 训练目标：学习噪声到专家动作的线性插值速度场。动作从均匀噪声ϵ ~ U([-1,1]^{50×24})出发，插值目标为真实动作a，时间t从Beta(2,2)分布采样（聚焦中间难度区域）。损失函数为L = E[||v_θ(x_t, t|...) - (a - ϵ)||^2]。

    5. **训练策略**：单阶段联合优化，VLM骨干和动作头同时训练（所有参数可微），使用DeepSpeed ZeRO-2和FP32主权重，避免BF16下的小权重更新丢失。学习率2e-5，1000步预热，余弦衰减，梯度裁剪1.0，权重衰减1e-4。
